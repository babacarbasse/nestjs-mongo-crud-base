### Usage

### Installation

```bash
npm i nestjs-mongo-crud-base
```

### Importing library

You can import the generated bundle to use the whole library generated by this starter:

```javascript
import {
  BaseController,
  BaseRepositoryService,
  BaseSchema
} from 'nestjs-mongo-crud-base';
```
